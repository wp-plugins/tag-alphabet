This plugin display all your post tags in vertical columns prefixed with alphabet letters.

== Description ==

This plugin display all your post tags in vertical columns prefixed with alphabet letters. After its installed you can use the following shortcode [tagalphabet columns=4] to display your post tags.

== Installation ==

1. Upload `essentials-tag-alphabet` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
